# 📌 CreditWise – Loan Approval Prediction System

An end-to-end supervised Machine Learning project to predict loan approval status using classification algorithms.

---

## 🔹 Project Overview

This project implements a binary classification model to determine whether a loan application should be approved based on applicant details such as income, credit history, loan amount, and other relevant features.

---

## 🔹 Features Implemented

- Data Cleaning and Preprocessing  
- Exploratory Data Analysis (EDA)  
- Feature Engineering  
- Model Training using:
  - K-Nearest Neighbors (KNN)
  - Logistic Regression
  - Naive Bayes
- Model Evaluation using:
  - Accuracy
  - Precision
  - Recall
  - F1 Score  

---

## 🔹 Tech Stack

- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  
- Scikit-learn  

---

## 🔹 Results

Among the implemented models, **Logistic Regression** achieved the best performance based on evaluation metrics.

---

## 🔹 Key Learnings

- Built a complete ML pipeline from data preprocessing to model evaluation  
- Gained hands-on experience with supervised learning algorithms  
- Improved understanding of classification metrics and model comparison
